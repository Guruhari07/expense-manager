# 💸 Expense Calculator PWA

A premium, cross-platform Progressive Web App (PWA) to help you track daily expenses, log income, and automatically manage your running balance right inside your browser.

## ✨ Features
* **Income & Expense Tracking:** Easily toggle between income and expenditure with dynamic color-coding.
* **Running Balances:** For every transaction added, the table tracks and displays a continuously updating mathematical balance.
* **Offline Functionality:** Built with a custom Service Worker, allowing the web app to function flawlessly even without an internet connection.
* **Installable (PWA):** Install it directly to your Android, iOS, or Desktop home screen like a native application without needing an app store!
* **Data Persistence:** Uses your browser's Local Storage so your data is saved automatically and never lost on page refresh.
* **Modern UI:** Built using modern styling, featuring sleek containers, gradient backgrounds, CSS variables, and fluid mobile responsiveness.

## 🛠️ Technology Stack
* Vanilla JavaScript
* HTML5
* CSS3

## 🚀 Getting Started
To run this project locally, you don't need any complex frameworks or package managers!
1. Clone the repository to your machine: 
```bash
git clone https://github.com/Guruhari07/expense-manager.git
```
2. Open the directory and simply launch `index.html` in any modern web browser.
3. Start tracking your expenses!

## 📱 Mobile Installation
If you deploy this app to a service like Netlify or Vercel:
1. Navigate to the live URL on your mobile device.
2. **Android:** Wait for the "Add to Home screen" pop-up at the bottom of the screen.
3. **iOS:** Tap the "Share" button and select "Add to Home Screen" from the menu.
4. The calculator will now look and operate natively as a standalone app on your device!
